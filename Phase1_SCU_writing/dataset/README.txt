Each year contains two SCU files to use in the AMT system summary evaluation task (phase 2).
The two files are the same, except for the 'forUse' field.
Each file signals 16 different SCUs per event, totaling 32 different SCUs to evaluate on (32 questions may be too much for a crowd worker).

2005:
    - 10 events (half) annotated
    - 494 relevant SCUs annotated
    - about 49 SCUs on average per event (for which 16+16 have been marked for use)
2006:
    - 20 events (all) annotated
    - 1021 relevant SCUs annotated
    - about 51 SCUs on average per event (for which 16+16 have been marked for use)
